Git alapparancsok

git init → Új Git repository inicializálása (helyi gépen).

git clone [repo-url] → Létező repository klónozása GitHub-ról vagy más távoli helyről.

git status → Megmutatja a fájlok jelenlegi állapotát (módosított, hozzáadott, commitolásra váró stb.).

git add [fájlnév] → Egy fájl hozzáadása a staging területhez.

git add . → Minden változás hozzáadása a staging területhez.

git commit -m "üzenet" → A staging területen lévő fájlok commitolása, üzenettel.

git log → A commit előzmények megtekintése.

git diff → Különbségek megtekintése a fájlok között (pl. staging előtt).

git branch → Az összes branch listázása.

git branch [új-branch-név] → Új branch létrehozása.

git checkout [branch-név] → Átváltás egy másik branch-re.

git checkout -b [új-branch-név] → Új branch létrehozása és azonnali váltás rá.

git merge [branch-név] → A megadott branch változásainak összeolvasztása a jelenlegi branch-be.

git remote add origin [url] → Távoli GitHub repository hozzáadása.

git push -u origin main → A main branch feltöltése GitHub-ra (első alkalommal -u).

git push → Változások feltöltése a távoli repository-ba.

git pull → A legfrissebb változások letöltése a GitHub-ról.
